Basic Agentic Code Review Service


## Commands

> npm run dev
>
> npm run start
> 
> npm run test
> 
> npm run clean && npm run build


## Requirements

- Node.js (project developed on v21)
- Docker (used to run scc static code analyzer)
- Ollama (local or accessible via URL endpoint)
